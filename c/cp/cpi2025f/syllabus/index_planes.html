Unicode 字元平面 Plane

Unicode 分為 17 平面（Plane），每平面擁有 65,536（2^16）個代碼點。

  1. U+0000 - U+FFFF 基本多文種平面 Basic Multilingual Plane (BMP)
  2. U+10000 - U+1FFFF 第一輔助平面 (多文種補充平面 ) Supplementary Multilingual Plane (SMP)
  3. U+20000 - U+2FFFF 第二輔助平面 (表意文字補充平面) Supplementary Ideographic Plane (SIP)
  4. U+30000 - U+3FFFF 第三輔助平面 (表意文字第三平面) Tertiary Ideographic Plane (TIP)
  5. U+40000 - U+5FFFF （尚未使用）
  6. U+50000 - U+6FFFF （尚未使用）
  7. U+60000 - U+7FFFF （尚未使用）
  8. U+70000 - U+8FFFF （尚未使用）
  9. U+80000 - U+9FFFF （尚未使用）
  10. U+90000 - U+AFFFF （尚未使用）
  11. U+A0000 - U+BFFFF （尚未使用）
  12. U+B0000 - U+CFFFF （尚未使用）
  13. U+C0000 - U+DFFFF （尚未使用）
  14. U+D0000 - U+DFFFF （尚未使用）
  15. U+E0000 - U+EFFFF 特別用途補充平面 Supplementary Special-purpose Plane (SSP)
  16. U+F0000 - U+FFFFF 保留作為私人使用區（A區）[1] Private Use Area-A (PUA-A)
  17. U+100000 - U+10FFFF 保留作為私人使用區（B區）[1] Private Use Area-B (PUA-B)

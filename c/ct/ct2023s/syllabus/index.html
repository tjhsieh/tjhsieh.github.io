• WRITES 守寫字 「守」住你的靈魂 「寫」出你的心情 「字」給你愛的每一個人 用你手心的溫度，來溫暖世界
• 稿紙造字小訣竅大分享 說起【稿紙造字】，應該是守寫字平台上最早登場的造字方式吧。早在守寫字的前一代(fontwalker)的前一代(fontcloud)就已經存在類似的流程，提供使用者建立自己的字體，但其影像處理技術方面還有不少需要克服的難關，所以同時也是最讓我們的開發團隊最頭痛的造字方式......
• 守寫字FB社團 現在的稿紙造字比起剛登場時其實已經修正了不少問題，但在所需的器材上依舊有很大的限制，原本以為掃描器會是最麻煩的門檻，但沒想到經過我們的調查後發現，彩色列印才是打退大部分人的最大難關。 ... 稿紙造字是透過圖片中"紅色"格線所組成的格子來切割使用者上傳的稿紙圖檔，因此需要彩色印表機來印出紅色格線的稿紙，並使用彩色掃描來使稿紙圖檔保持格線的色彩，但反過來說，只要"最後上傳的圖檔"是紅色格線的，中間的過程是怎樣就不重要了！ ... 用修圖的概念來把黑白的掃描圖檔加上紅色格線。 ... 而加入紅色格線的方式選擇上，可以直接使用小畫家等基礎繪圖工具，打開圖檔後藉由直線工具達成。
• 致力於手寫文字的文藝復興者 是什麼那麼重要，可以在過去三年裡，讓三個碩士、一個博士畢業、一個工程師變成專案經理？起初我們幾個資工男孩開始著手中研院的「國家數位典藏技術研發」計劃時，並沒有太大的熱情，我們不懂中文與歷史的美，總認為「科技」才是王道，但不知不覺被這個計畫霸王硬上弓了，我們看到看到康熙真跡、胡適手稿，被這些文字的美感動，就像有一股暖流在身體流動，「熱熱的…….刺刺的......」，所以這是我們的使命，要讓「守寫字」這股暖流，流到每個手寫繁體字的人心裡，不要讓他的好只有我們幾個資工男孩知道。
• 我們花費三年、1095天、26280小時所累積的技術，可以在未來創造出無限的可能， 只要你們認同我們的理念，就用你們的手，守住這個理念，一起寫出寫的可能……
